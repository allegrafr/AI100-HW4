# AI100-HW4
# AI 100 Homework 4 - Ollama API Interaction

## Requirements
- Python 3.x
- `requests` library (install with `pip install requests`)

## How to Use

1. Make sure the Ollama server is running locally at `http://localhost:11434`.
2. Run the Python script with:
   ```bash
   python ollama_interaction.py
   ```

## Purpose
This script interacts with the Ollama API to send prompts to a local model and receive responses.
